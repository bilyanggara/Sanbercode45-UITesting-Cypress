## Run Locally

Clone the project

```bash
  git clone https://github.com/bilyanggara/Sanbercode45-UITesting-Cypress.git
```

Go to the project directory

```bash
  cd project-name
```

Install dependencies

```bash
  npm install
```

Start the project

```bash
  npm run start
```


Open Cypress

```bash
  npm run Open
```
